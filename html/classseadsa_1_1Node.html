◆ mergeAllocSites() [1/2]

unsigned seadsa::Node::mergeAllocSites ( Node &  n)

compute a simulation relation between this and n while propagating allocation sites for all the nodes in the relation. The result can only be one of these four values: 0x0 (no change), 0x1 (this changed), 0x2 (n changed), 0x3 (both changed).
